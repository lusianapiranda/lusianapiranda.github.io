My Introduction

Perkenalkan nama saya Lusiana Nathalia Piranda dari jurusan TI di USTJ angkatan 20 dengan npm 20411022. Saya orang Makassar dan lahir di Makassar, 16 Juni 2002 dan sejak smp sudah pindah ke Papua karena ayah pindah kerja. Tujuan saya membuat website ini untuk memenuhi tugas besar mata kuliah Multimedia dalam membuat website portofolio dengan project Praktikum Multimedia. Dalam membuat desain web saya telah berpengalaman semenjak kuliah di jurusan TI dan masih terus belajar. Sekian dan Terima Kasih.
